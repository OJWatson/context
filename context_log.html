Start and stop log

Usage

context_log_start()


context_log_stop()


parse_context_log(x)

Arguments

x
Vector of log output

Description

Start and stop the context log. Soon this might swap out for loggr, but for now this should do. When active, some actions will print diagnostic information to the message stream. This is used in particular by the program installed by install_context.

Details

The interface here will change by adding arguments. Future versions may support logging to a file.
