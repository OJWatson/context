Save and reload tasks

Usage

task_save(expr, context, envir = parent.frame())


task_save_list(list, context, envir = parent.frame())


task_load(handle, install = TRUE, envir = .GlobalEnv)

Arguments

expr
An expression to save
context
Either a context or a context_handle object. If missing (or NULL) one will be automatically generated, using envir and root. See context_save.
envir
Passed through to context_save when locating local variables.
list
a list of tasks, all to be evaluted in the same context.
handle
A handle to load the task
install
Should missing packages be installed?
root
root

Description

Save and reload tasks. Tasks consist of an expression bound to a context.
