Install script

Usage

install_context(path)

Arguments

path
Path to install the script, ideally on your $PATH. This directory must exist.

Description

Install a script that runs the context. This won't work on windows, but I'll get that sorted soon.

Details

This script requires that context (and its dependencies are installed in a library that is readable on startup (e.g., the user or site lib). If it is not installed, then this script will fail.
